# ShortURL

This is a simple URL shortener written in Clojure.
I've followed this [tutorial](https://www.youtube.com/watch?v=0mrguRPgCzI&t=3s) to build it.

- Sign up or sign in on planetscale, and create a db.
- Create env.edn file with a map of the db variables.

Example of env.edn file:
```
 {:HOST ""
  :DBTYPE ""
  :DBNAME ""
  :USER ""
  :PASSWORD ""}
 ```
